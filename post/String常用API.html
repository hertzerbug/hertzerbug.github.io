字符串的内容比较

  1. 示例：

    package string;

import java.util.Scanner;

public class StringEqualDemo4 {
    public static void main(String[] args) {
        //1. 正确登录名
        String oknames = "张三";
        String okpassword = "123456";

        //2.请您输入登录名称和密码
        Scanner sc = new Scanner(System.in);
        System.out.println("登录名称");
        String name = sc.next();
        System.out.println("登陆密码");
        String password = sc.next();

        //3.判断用户输入的登录名称和密码是否正确
        if (name == oknames && okpassword == password){
            System.out.println("登录成功");
        }else{
            System.out.println("用户名或密码错误！");
        }
    }
}

    运行结果：

    登录名称
张三
登陆密码
123456
用户名或密码错误！

    结论：字符串的内容比较不适合用“==”比较。

  2. 推荐使用String类提供的”equals”比较：只关心内容一样。

  • 方法名：boolean equals(Object anObject)
    将此字符串与指定对象进行比较，只关心字符内容是否一致。
  • 方法名：boolean equalsIgnoreCase(String anotherString)
    将此字符串与指定内容进行比较，忽略大小写比较字符串，只关心字符串内容是否一致！
    示例：
    package string;

import java.util.Objects;
import java.util.Scanner;

public class StringEqualDemo4 {
    public static void main(String[] args) {
        //1. 正确登录名
        String oknames = "张三";
        String okpassword = "123456";

        //2.请您输入登录名称和密码
        Scanner sc = new Scanner(System.in);
        System.out.println("登录名称");
        String name = sc.next();
        System.out.println("登陆密码");
        String password = sc.next();

        //3.判断用户输入的登录名称和密码是否正确
        if (name.equals(oknames) && okpassword.equals(password)){
            System.out.println("登录成功");
        }else{
            System.out.println("用户名或密码错误！");
        }
        //4.忽略大小写内容的API：一般用于比较验证码这样的业务逻辑。
        String syscode = "23AdFh";
        String code1 = "23adfh";
        System.out.println(syscode.equals(code1));//false
        System.out.println(syscode.equalsIgnoreCase(code1));//true

    }
}
    结果：
    登录名称
张三
登陆密码
123456
登录成功
false
true
  1. 总结
    （1）如果是字符串比较应该使用什么方式进行比较？
    • 使用String提供的equlas方法。
    • 只关心内容一样就返回true。
      (2)开发中什么时候使用==比较数据
    • 基本数据类型比较时使用。

String常用API-遍历、替换、截取、分割操作

  1. String常用API
方法名 说明
int length() 返回此字符串的长度
char charActer() 获取某个索引位置的字符
char[ ] toCharArray(): 将当前字符串转换成字符数组返回
String substring(int beginIndex, int endIndex) 根据开始和结束索引进行截取，得到新的字符串（不包括前后）
String substring(int beginIndex) 从传入的索引处截取，截取到末尾，得到新的字符串
String replace(CharSequence target, CharSequence replacement) 使用新值，将字符串中的旧值替换，得到新的字符串
String[ ] split(String regex) 根据传入的规则切割字符串，得到字符串数组返回
boolean contains(CharSequence s) 判断字符串里是否包含关键字
boolean startsWith(String prefix, int toffset) 判断字符串以什么开始
  1. 示例
    package string;

public class StringAPIOtherDemo5 {
    public static void main(String[] args) {
        //1.int length()|返回此字符串的长度|
        String name = "我爱你中国love";
        System.out.println(name.length());

        //2.char charAt()|获取某个索引位置的字符|
        char c = name.charAt(1);
        System.out.println(c);

        System.out.println("----遍历字符串中的每个字符----------");
        for (int i = 0; i < name.length(); i++) {
            char ch = name.charAt(i);
            System.out.println(ch);
        }

        //3.char[ ] toCharArray():|将当前字符串转换成字符数组返回|
        char[] chars = name.toCharArray();
        for (int i = 0; i < chars.length; i++) {
            char ch = chars[i];
            System.out.println(ch);
        }

        //4.String substring(int beginIndex, int endIndex)|根据开始和结束索引进行截取，得到新的字符串（不包括前后）|
        String name2 = "java是最厉害的编程语言！";
        String rs = name2.substring(0,9);
        System.out.println(rs);
        String rs1 = name2.substring(4,9);
        System.out.println(rs1);

        //5.String substring(int beginIndex)|从传入的索引处截取，截取到末尾，得到新的字符串|
        String rs2 = name2.substring(4);
        System.out.println(rs2);

        //6.String replace(CharSequence target, CharSequence replacement)|使用新值，将字符串中的旧值替换，得到新的字符串|
        String name3 = "张三是最厉害的80后，张三棒棒的！我好爱张三";
        String rs3 = name3.replace("张三","**");
        System.out.println(rs3);

        //7.boolean contains(CharSequence s)|判断字符串里是否包含关键字|
        System.out.println(name3.contains("张三"));//true
        System.out.println(name3.contains("张二"));//false

        //8.|boolean startsWith(String prefix, int toffset)|判断字符串以什么开始|
        System.out.println(name3.startsWith("张三"));//true
        System.out.println(name3.startsWith("张三是最厉害的"));//true
        System.out.println(name3.startsWith("张三2"));//false

        //9.String[ ] split(String regex)|根据传入的规则切割字符串，得到字符串数组返回|
        String name4 = "王宝强，贾乃亮，谢霆锋";
        String[] names = name4.split("，");
        for (int i = 0; i < names.length; i++) {
            System.out.println("选择了：" + names[i]);
        }

    }
}

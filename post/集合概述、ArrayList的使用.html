集合概述、ArrayList的使用

集合概述

  1. 集合与数组类似，也是一种容器，用于装数据的。
  2. 数组的特点
    • 数组定义完成启动之后，类型确定，长度固定。
    • 问题：在个数不能确定，且要进行增删数据操作的时候，数组是不大合适的。
  3. 集合的特点
    • 集合的大小不固定，启动之后可以动态变化，类型也可以选择不固定。
    • 集合非常适合做元素个数不确定，且要进行增删操作的业务场景。
    • 集合的提供了很多功能，而数组功能单一。
  4. 总结
    （1）数组和集合的元素储存的个数问题？
    • 数组定义后类型确定，长度固定
    • 集合类型可以不固定，大小是可变的。
      （2)数组和集合适合的场景
    • 数组适合做数据个数和类型确定的场景
    • 集合适合做数据个数不确定，且要做增删元素的场景。

ArrayList集合

  1. ArrayList集合
    • ArrayList是集合中的一种，它支持索引。
  2. ArrayList集合的对象获取
构造器 说明
public ArrayList() 创建一个空的集合对象
  1. ArrayList集合添加元素的方法
构造器 说明
boolean add(E e) 将指定元素追加到此集合的末尾
void add(int index, E element) 在此集合中的指定位置插入指定的元素
4. 示例 
package arraylist;

import java.util.ArrayList;

public class ArrayListDemo1 {
    public static void main(String[] args) {
        //1.创建ArrayList集合的对象
        ArrayList list = new ArrayList();

        //2.添加数据
        list.add("java");
        list.add("java");
        list.add("MySQL");
        list.add("张三");
        list.add(23);
        list.add(23.5);
        list.add(false);
        list.add('中');

        System.out.println(list);

        //3.给指定索引位置插入元素
        list.add(1,"李四");
        System.out.println(list);
    }
}

运行结果

[java, java, MySQL, 张三, 23, 23.5, false, 中]
[java, 李四, java, MySQL, 张三, 23, 23.5, false, 中]
  1. 总结
    （1）Arraylist类如何创建集合对象的，如何添加元素？
    • Arraylist list = new ArrayList()
    • public boolean add(E,e)
    • public void add(int index,E element)

ArrayList对于泛型的支持

1.泛型概述

  • Arraylist<E>: 其实就是一个泛型类,可以在编译阶段约束集合对象只能操作某种数据类型
  1. 举例：
  • ArrayList<String>:此集合只能操作字符串类型的元素。
  • ArrayList<Integr>:此集合只能操作整数类型的元素。
    注意：集合中只能存储引用类型，不支持基本数据类型。
  1. 示例：
    package arraylist;

import java.util.ArrayList;

public class ArrayListDemo2 {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<>();//字符串类型
        list.add("java");
        list.add("java");
        list.add("MySQL");
        list.add("张三");

        ArrayList<Integer> list2 = new ArrayList<>();//整型
        list2.add(12);
        list2.add(34);
        list2.add(56);
        
        ArrayList<Object> list3 = new ArrayList<>();//所有类型
    }
}
  2. 总结
    （1）怎么去统一ArrayList集合操作的元素类型？
    • 使用泛型：<数据类型>
    • ArrayList list = new ArrayList();

ArrayList常用API、遍历

  1. ArrayList集合常用方法
方法名称 说明
E get(int index) 返回指定索引处的元素
int size() 返回集合中的元素个数
E remove(int index) 删除指定索引处的元素，返回被删除的元素
boolean remove(Object o) 删除指定的元素，返回删除是否成功
E set(int index, E element) 修改指定索引处的元素，返回被修改的元素。
2. 示例 
package arraylist;

import java.util.ArrayList;

public class ArrayListDemo3 {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<>();
        list.add("java");
        list.add("java");
        list.add("MySOL");
        list.add("MyBatis");
        list.add("HTML");

        //1.E get(int index)|返回指定索引处的元素
        String e = list.get(3);
        System.out.println(e);

        //2.int size()|返回集合中的元素个数
        System.out.println(list.size());

        //3.完成元素的遍历
        for (int i = 0; i < list.size(); i++) {
            System.out.println(list.get(i));
        }

        //4.E remove(int index)|删除指定索引处的元素，返回被删除的元素
        System.out.println(list);//[java,java,MySQL,MyBatis,HTML]
        String e2 = list.remove(2);
        System.out.println(e2);
        System.out.println(list);

        //5.boolean remove(Object o)|删除指定的元素，返回删除是否成功,删除成功返回true，删除失败返回false。
        System.out.println(list.remove("MyBatis"));
        System.out.println(list);
        //只会删除第一个出现的元素值
        System.out.println(list.remove("java"));
        System.out.println(list);

        //6.E set(int index, E element)|修改指定索引处的元素，返回被修改的元素。
        String e3 = list.set(0,"C++");
        System.out.println(e3);
        System.out.println(list);
    }
}

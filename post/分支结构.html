分支结构

  1. if分支，根据判定的结果（真或假）决定执行某个分支

if功能强于switch
if适合做区间匹配

  • 格式1：

    if (条件表达式){
    语句体;
}

    例：

    //需求：心跳（60-100）之间是正常的，否则系统进一步检查
   //格式1
   Scanner sc = new Scanner(System.in);
   System.out.println("请输入您的年龄：");
   int heartBate = sc.nextInt();
   if (heartBate < 100 || heartBate > 60) {
       System.out.println("心跳正常");
   }
   System.out.println("检查结束");

  • 格式2

    if (条件表达式){
    语句体1;
} else {
    语句体2;
}

  • 格式3

    if(条件表达式1){
    语句体1;
} else if(条件表达式2){
    语句体2;
} else if (条件表达式2){
    语句体2;
} 
...
else {
    语句体n+1;
}
  1. switch分支
  • 适合做匹配的分支选择，结构清晰，格式良好。

switch适合做值匹配

switch(表达式){
    case 值1 :
       //语句
       break; //可选
    case 值2 :
       //语句
       break; //可选
    //你可以有任意数量的case语句
    case 值n-1 ：
        break;
    default : //可选
       //执行第n条语句
}

例：

package com.itheima.branch;

import java.util.Scanner;

public class SwitchDemo2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("请输入您的等级：");
        char grade = sc.next().charAt(0);//Scanner键盘输入如何赋值给char类型变量
        switch(grade)
        {
            case 'A':
                System.out.println("优秀");
                break;
            case 'B' :
            case 'C' :
                System.out.println("良好");
                break;
            case 'D' :
                System.out.println("及格");
                break;
            case 'F' :
                System.out.println("你需要再努力努力");
                break;
            default :
                System.out.println("未知等级");
        }
        System.out.println("你的等级是 " + grade);
    }

    private static class string {
    }
}
  • 注意事项：
    • 表达式类型只能是 byte、short、int、char，JDK5开始支持枚举，JDK7开始支持String、不支持double、float、long。
    • case给出的值不允许重复，且只能是字面量，不能是变量。
    • 不能忘记写break，会发生穿透。

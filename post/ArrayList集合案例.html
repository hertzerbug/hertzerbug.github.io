案例：遍历并删除元素

  1. 需求
  • 某个班级的考试在系统上进行，成绩大致为：98，77，66，89，79，50，100
  • 现在需要先把成绩低于80分以下的数据去掉。
  1. 分析：
    （1）定义ArrayList集合储存多名学员的成绩
    （2)遍历集合每个元素，如果元素低于80分，去掉他。
  2. 代码实现
    package arraylist;

import java.util.ArrayList;

public class Text1 {
    public static void main(String[] args) {
        //遍历并删除元素
        //(98，77，66，89，79，50，100)
        ArrayList<Integer> list = new ArrayList<>();
        list.add(98);
        list.add(77);
        list.add(66);
        list.add(89);
        list.add(79);
        list.add(50);
        list.add(100);
        System.out.println(list);

        //2.把80分以下的成绩从集合中去掉。
        
        /**
         * 完美方案之一
        for (int i = 0; i < list.size(); i++) {
            int l = list.get(i);
            if (l < 80) {
                //这个分数必须删除
                list.remove(i);
                i--;//删除成功之后，必须退一步保证到这个位置。
            }
        }
        System.out.println(list);
         */
        
        //完美方案之二，倒序遍历删除；
        for (int i = list.size() - 1; i >=0 ; i--) {
            int l = list.get(i);
            if (l < 80) {
                //这个分数必须删除
                list.remove(i);
            }
        }
        System.out.println(list);
    }
}
    实现结果
    [98, 77, 66, 89, 79, 50, 100]
[98, 89, 100]

案例：存储自定义类型的对象

  1. 需求
  • 某院系影片需要在后台存储上述三部电影，然后依次展示出来。
  1. 分析
    （1）定义一个电影类，定义一个集合存储电影对象。
    （2）创建3个电影对象，封装相关数据，把3个对象存入到集合中去。
    （3）遍历集合中的3个对象，输出相关信息。
  2. 代码实现：
    arraylist;
    
import java.util.ArrayList;

public class Text5 {
    public static void main(String[] args) {
        //1.定义电影类。
        //1.创建电影形象
        Movie m1 = new Movie("《肖申克的救赎》",9.7,"罗宾斯");
        Movie m2 = new Movie("《霸王别姬》",9.6,"张国荣");
        Movie m3 = new Movie("《阿甘正传》",9.5,"汤姆");

        //3.创建一个电影类型的ArrayList集合，存储三部电影对象
        ArrayList<Movie> movies = new ArrayList<>();
        movies.add(m1);
        movies.add(m2);
        movies.add(m3);
        System.out.println(movies);
        //4.遍历电影类型的集合中的每个电影对象
        for (int i = 0; i < movies.size(); i++) {
            Movie m = movies.get(i);
            System.out.println(m.getName() + m.getScore() + m.getActor());
            System.out.println("-------------------");
        }
    }
}

案例：学生信息系统的数据搜索

  1. 需求
  • 后台程序需要存储加上学生信息并展示，然后要提供按照学号搜索学生信息的功能。
  1. 分析
    （1）定义一个Student类，定义ArrayList集合如上学生信息，并遍历展示出来。
    （2）提供一个方法，可以接收ArrayList集合，和要搜索的学号，返回搜索到的学生对象信息，并展示。
    （3)使用死循环，让用户可以不停的搜索。
  2. 代码实现
    package arraylist;

import java.util.ArrayList;
import java.util.Scanner;

public class Text6 {
    public static void main(String[] args) {
        //1.定义学生类
        //2.创建集合存储学生对象，创建学生对象封装学生对象
        Student stu1 = new Student("20180302","张三",23,"护理一班");
        Student stu2 = new Student("20180303","李四",23,"推拿二班");
        Student stu3 = new Student("20180304","王二",23,"中药学一班");
        Student stu4 = new Student("20180305","武大",23,"神经科四班");
        ArrayList<Student> students = new ArrayList<>();
        students.add(stu1);
        students.add(stu2);
        students.add(stu3);
        students.add(stu4);
        //4.遍历学生信息展示
        System.out.println("学号\t\t\t名称\t年龄\t班级");
        for (int i = 0; i < students.size(); i++) {
            Student s = students.get(i);
            System.out.println(s.getStudyNumber() + "\t" + s.getName() + "\t" + s.getAge() + "\t" + s.getClassName());
        }
        //4.定义方法完成学号的搜索功能。
        Scanner sc = new Scanner(System.in);
        while(true){
            System.out.println("请输入学号：");
            String studyNumber = sc.next();
            //5.调用方法查询
            Student s = getStudentById(students,studyNumber);
            //6.判断s中是否存在学生对象
            if (s == null){
                System.out.println("查无此人");
            }else{
                System.out.println("你查询的信息如下");
                System.out.println("学号\t\t\t名称\t年龄\t班级");
                System.out.println(s.getStudyNumber() + "\t" + s.getName() + "\t" + s.getAge() + "\t" + s.getClassName());
            }
        }
    }

    /**
     * 根据学生学号查询学生对象并返回
     * @param student 存储全部学生对象的集合
     * @param studyNumber 搜索学生的学号
     * @return 学生对象 | null
     */
    public static Student getStudentById(ArrayList<Student> student,String studyNumber){
        //1.遍历全部学生对象
        for (int i = 0; i < student.size(); i++) {
            Student s = student.get(i);
            //2询问查找的学好是否是我们要找到学号
            if(s.getStudyNumber().equals(studyNumber)){
                return s;
            }
        }
        //查无此人
        return null;
    }
}
    Student类代码

        this.className = className;
    }

}

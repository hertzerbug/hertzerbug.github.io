面向对象

面向对象概述

  • 面向：找、拿。
  • 对象：东西。
  • 面向对象编程：找或拿东西过来编程。
  1. 面向对象编程的好处
    符合人类思维习惯，编程更简单、更好理解。
  2. 面向对象学什么
    学习自己设计对象并使用 —> 必须掌握面向对象的语法

设计对象并使用

  1. 类是什么
    类（设计图）：是对象共同特征的描述；
    对象：是真实存在的具体案例。
    结论：在Java中必须先设计类，才能创建的对象并使用。
  2. 如何设计类
    pubilc class 类名{
    1.成员变量（代表属性，一般是名词）
    2.成员方法（代表行为，一般是动词）
    3.构造器
    4.代码块
    5.内部类
}
    pubilc class Car{
    //属性（成员变量）
    String name;
    double price;
    //行为（方法）
    public void start(){

    }
    pubilc void run(){

    }
}
  3. 如何得到类的对象
    类名 对象名 = new 类名();
Car c = Car();
  4. 如何使用对象 
    * 访问属性：对象名，成员变量
* 访问行为：对象名.方法名(...)
  5. 示例：
    package createobject;

定义类的补充注意事项

  • 类名首字母建议大写，且有意义，满足驼峰模式。
  • 一个Java文件中可以定义多个class类，但只能一个类是public修饰，而且public修饰的类名必须成为代码文件名。
    实际开放中建议还是一个文件定义一个class类。
  • 成员变量的完整定义格式是：修饰符 数据类型 变量名称 = 初始化值；一般无需指定初始化值，存在默认值。

    package memory;

/**
 * 理解；两个变量指向同一个对象的内存运行机制
 */

public class Text {
    public static void main(String[] args) {
        //创建学生对象
        Student s1 = new Student();
        s1.name = "小明";
        s1.sex = '男';
        s1.hobby = "吃饭睡觉学习";
        s1.study();

        //把s1的值赋值给一个学生类型的变量s2。
        Student s2 = s1;
        System.out.println(s1);//对象地址：memory.Student@4cb2c100
        System.out.println(s2);//对象地址：memory.Student@4cb2c100

        s2.hobby = "爱提问";
        System.out.println(s2.name);
        System.out.println(s2.sex);
        System.out.println(s1.hobby);//结果是：爱提问
        System.out.println(s2.hobby);//结果是：爱提问
    }
}
    package memory;

public class Student {
    String name;
    char sex;
    String hobby;

    public void study(){
        System.out.println("名称：" + name + "，性别：" + sex + "，爱好：" + hobby + "开始学习了");
    }

}

项目：ATM系统

银行ATM系统技术选型分析：

  1. 技术点分析
    （1）面向对象编程
    每个用户账户都是一个对象：所以需要设计账户类Account用于创建账户对象封装用户信息。
    （2）使用集合容器
    系统需要提供一个容器用于储存这些账户对象的信息，我们选ArrayList集合。
    （3）程序流程控制
    需要集合分支、循环跳转关键字等相关操作控制程序的业务逻辑。
    （4）使用常见API
    登录信息的内容比较，业务数据的分析、处理等都需要用到String等常用API来解决。

系统准备、首页设计

  1. 系统准备内容设计：
    （1）美俄给用户的账户信息都是一个对象，需要提供账户类。
    （2）需要准备一个容器，用于存储系统全部账户信息。
    （3）首页只需要包含：注册与登陆两个功能。
  2. 实现步骤
    （1）定义账户类，用于后期创建账户对象封装用户的账户信息。
    （2）账户类中的信息至少包含（卡号、姓名、密码、余额、取现额度）。
    （3）需要准白一个ArrayList的集合，用于存储用户的账户对象。
    （4）需要展示欢迎页饱含2个功能：开户功能、登录账户。
  3. 实现代码：

用户开户功能实现

  1. 分析
    （1）开户功能其实就是往系统的集合容器存入一个账户对象的信息。
  2. 开户功能实现步骤
    （1）开户应该定义成一个方法，并传入账户集合：
    pubilc static viod register (ArrayList<Account>accounts){...}
    （2）创建一个账户类的对象用于封装账户信息（姓名、密码、卡号）
    （3）键盘录入姓名、密码、确认密码（需保证两次密码一致）。
    （4）生成账户卡号，卡号必须由系统生成8位数字（必须保证卡号的唯一）
    （5）把Account账户对象存入集合accounts中去。
  3. 代码实现：

/**
 * 方法用户开户功能的实现
 *
 * @param accounts 接收的账户集合
 */
private static void register(ArrayList<Account> accounts, Scanner sc) {
    System.out.println("===============系统开户操作===============");
    //1.创建一个账户对象，用于后期封装账户信息。
    Account account = new Account();

    //2.录入当前账户信息
    System.out.println("请输入账户用户名");
    String userName = sc.next();
    account.setUserName(userName);

    while (true) {
        System.out.println("请输入账户密码");
        String passWord = sc.next();
        account.setUserName(userName);
        System.out.println("请确认账户密码");
        String okpassWord = sc.next();
        account.setUserName(userName);
        if (passWord.equals(okpassWord)) {
            //密码认证通过
            account.setPassWord(passWord);
            break;
        } else {
            System.out.println("你输入的两次密码不同，请重新确认！");
        }
    }

    System.out.println("请输入账户当次限额：");
    double qutaMoney = sc.nextDouble();
    account.setQuotaMoney(qutaMoney);
    //为账户随机8位账户卡号(独立功能独立成方法)
    String cardId = getRandCardId(accounts);
    account.setCarId(cardId);

    //3.把账户对象添加到账户集合中去
    accounts.add(account);
    System.out.println("恭喜" + userName + "开户完成" + "你的卡号是：" + cardId + "，请妥善保管");
}

/**
 * 生成8位随机卡号
 *
 * @return
 */
private static String getRandCardId(ArrayList<Account> accounts) {
    //1.生成8位数字
    Random r = new Random();
    while (true) {
        String carId = "";
        for (int i = 0; i < 8; i++) {
            carId += r.nextInt(10);
        }

        //2.判断这个8位的卡号是否和其他账户相同
        Account acc = getAccountByCarId(carId,accounts);
        if (acc == null){
            return carId;
        }
    }
}

/**
 * 验证呢卡号是否重复
 * @param carId
 * @param accounts
 * @return
 */
private static Account getAccountByCarId(String carId, ArrayList<Account> accounts) {
    for (int i = 0; i < accounts.size(); i++) {
        Account acc = accounts.get(i);
        if (carId.equals(acc.getCarId())) {
            return acc;
        }
    }
    return null;//查无此用户
}

存款功能实现

  1. 分析
    （1）存款就是拿到当前账户对象。
    （2）然后让账户存入存款的金额。
    （3）调用账户对象的setMoney方法将账户余额修改成存钱后的余额。
    （4）存钱之后查询以下账户信息，确认存钱成功。
  2. 代码实现
    /**
 * 存钱
 * @param acc
 * @param sc
 */
private static void depositMoney(Account acc, Scanner sc) {
    System.out.println("===============用户存钱操作===============");
    System.out.println("请输入存款金额：");
    double money = sc.nextDouble();
    
    //更新账户余额原来的钱加上存入的钱
    acc.setMoney(acc.getMoney() + money);
    System.out.println("存款成功，你的账户信息如下：");
    showAccount(acc);
}

取款功能实现

  1. 分析
    （1）取款前先判断账户是否有钱
    （2）有钱则拿到自己账户对象
    （3）然后让用户输入取款金额
    （4）判断取款金额是否超过了当次限额，以及余额是否足够。
    （5）满足要求则调用账户对象的setMoney方法完成金额的修改。
  2. 代码实现
    /**
 * 取款功能
 *
 * @param acc
 * @param sc
 */
private static void drawMoney(Account acc, Scanner sc) {
        System.out.println("=================用户取钱功能=================");
        //1.判断账户是否足够100元
        if (acc.getMoney() < 100) {
            System.out.println("当前账户余额不足100元，不能取钱。");
            return;
        }
        while (true) {
        //2.提示用户输入取款金额
        System.out.println("请你输入取款额度");
        double money = sc.nextDouble();

        //3.判断金额是否超出存款
        if (acc.getMoney() >= money) {
            if (acc.getQuotaMoney() < money){
                System.out.println("你取款的额度大于当次限额" + acc.getQuotaMoney());
            }else{
                acc.setMoney(acc.getMoney() - money);
                System.out.println("你取款" + money + "成功！");
                showAccount(acc);
                return;
            }
        }else {
            System.out.println("你的额度不足，当前额度：" + acc.getMoney());
        }
    }
}

用户转账功能实现

  1. 分析
    （1）转账功能需要判断系统中是否有2个以上账户对象。
    （2）同时还要判断自己账户是否有钱。
    （3）输入对方卡号，判断对方账户是否存在。
    （4）对方账户存在还需要认证对方账户的姓氏。
    （5）满足要求则可以把自己账户对象的金额修改到其他账户对象中去。
  2. 代码实现：
    /**
 * 转账功能
 *
 * @param acc
 * @param sc
 * @param accounts
 */
private static void transferMoney(Account acc, Scanner sc, ArrayList<Account> accounts) {
    System.out.println("=================用户转账操作=================");
    //1.判断账户是否有两个
    if (accounts.size() < 2) {
        System.out.println("当前账户少于2个，请开户");
        return;
    }

    if (acc.getMoney() == 0) {
        //判断账户是否有钱
        System.out.println("你的账户没有钱！无法转账");
        return;
    }

    while (true) {
        //3.开始转账
        System.out.println("请输入对方卡号：");
        String cardId = sc.next();

        //这个卡号不能是自己的卡号
        if (cardId.equals(acc.getCarId())){
            System.out.println("不能给自己转账！");
            continue;//循环进入下一次
        }
        Account account = getAccountByCarId(cardId, accounts);
        //判断卡号是否存在
        if (account == null){
            System.out.println("你输入的账户不存在！请重新输入");
        }else{
            //该账户对象存在
            String userName = account.getUserName();
            String tip = "*" + userName.substring(1);
            System.out.println("请输入" + tip +"的姓氏：");
            String preName = sc.next();

            //判断姓氏是否正确
            if (preName.equals(userName.substring(0,1))){
                //认证通过
                //开始转帐，输入转账金额
                while (true) {
                    System.out.println("请输入转账金额：");
                    double money = sc.nextDouble();
                    //判断转账金额是否足够
                    if (money < acc.getMoney()){
                        acc.setMoney(acc.getMoney() - money);
                        account.setMoney(account.getMoney() + money);
                        System.out.println("账户" + acc.getUserName() + "成功给账户" + account.getUserName() + "转入金额" + money+ "元！");
                        return;//直接结束方法
                    }else{
                        System.out.println("你输入的余额不足" + money + "元!");
                    }
                }
            }else{
                System.out.println("对不起，你输入的信息有误！");
            }
        }
    }
}

                case 1:
                    //查询账户
                    showAccount(acc);
                    break;
                case 2:
                    //存款
                    depositMoney(acc, sc);
                    break;
                case 3:
                    //取款
                    drawMoney(acc, sc);
                    break;
                case 4:
                    //转账
                    transferMoney(acc, sc, accounts);
                    break;
                case 5:
                    //修改密码
                    updatePassWord(acc,sc);
                    return;
                case 6:
                    //退出
                    System.out.println("退出成功，欢迎下次光临！");
                    return;
                case 7:
                    //注销账户
                    if(daleteAccount(sc,acc,accounts)) {
                        return;//销户成功，返回首页
                    }else{
                        break;//销户不成功
                    }
                default:
                    System.out.println("你输入的命令不正确");
            }
        }
    }

    /**
     * 销毁账户
     * @param sc
     * @param acc
     * @param accounts
     */
    private static boolean daleteAccount(Scanner sc, Account acc, ArrayList<Account> accounts) {
        //从当前账户集合中删除账户对象，销毁就完成了
        System.out.println("你是否要销户? y/n");
        String rs = sc.next();
        if (rs.equals("y")){
            if (acc.getMoney() > 0){
                System.out.println("账户中的余额还未取完，不能销户。");
            }else {
                accounts.remove(acc);
                System.out.println("当前账户已经销除");
                return true;
            }
        }else{
            System.out.println("当前账户保留。");
        }
        return false;
    }


    /**
     * 修改密码
     * @param acc
     * @param sc
     */
    private static void updatePassWord(Account acc, Scanner sc) {
        System.out.println("=================用户密码修改=================");
        while (true) {
            System.out.println("请你输入当前密码：");
            String passWord = sc.next();
            //判断密码是否正确
            if (passWord.equals(acc.getPassWord())){
                while (true) {
                    System.out.println("请输入你要修改的密码：");
                    String updatePassWord = sc.next();

                    System.out.println("请确认你输入的密码：");
                    String okupdatePassWord = sc.next();

                    if (updatePassWord.equals(okupdatePassWord)){
                        acc.setPassWord(updatePassWord);
                        System.out.println("恭喜你，密码修改成功！");
                        return;
                    }else{
                        System.out.println("两次输入的密码不同！");
                    }
                }
            }else{
                System.out.println("你输入的密码错误，请输入密码：");
            }
        }
    }

    /**
     * 转账功能
     * @param acc
     * @param sc
     * @param accounts
     */
    private static void transferMoney(Account acc, Scanner sc, ArrayList<Account> accounts) {
        System.out.println("=================用户转账操作=================");
        //1.判断账户是否有两个
        if (accounts.size() < 2) {
            System.out.println("当前账户少于2个，请开户");
            return;
        }

        if (acc.getMoney() == 0) {
            //判断账户是否有钱
            System.out.println("你的账户没有钱！无法转账");
            return;
        }

        while (true) {
            //3.开始转账
            System.out.println("请输入对方卡号：");
            String cardId = sc.next();

            //这个卡号不能是自己的卡号
            if (cardId.equals(acc.getCarId())){
                System.out.println("不能给自己转账！");
                continue;//循环进入下一次
            }
            Account account = getAccountByCarId(cardId, accounts);
            //判断卡号是否存在
            if (account == null){
                System.out.println("你输入的账户不存在！请重新输入");
            }else{
                //该账户对象存在
                String userName = account.getUserName();
                String tip = "*" + userName.substring(1);
                System.out.println("请输入" + tip +"的姓氏：");
                String preName = sc.next();

                //判断姓氏是否正确
                if (preName.equals(userName.substring(0,1))){
                    //认证通过
                    //开始转帐，输入转账金额
                    while (true) {
                        System.out.println("请输入转账金额：");
                        double money = sc.nextDouble();
                        //判断转账金额是否足够
                        if (money < acc.getMoney()){
                            acc.setMoney(acc.getMoney() - money);
                            account.setMoney(account.getMoney() + money);
                            System.out.println("账户" + acc.getUserName() + "成功给账户" + account.getUserName() + "转入金额" + money+ "元！");
                            return;//直接结束方法
                        }else{
                            System.out.println("你输入的余额不足" + money + "元!");
                        }
                    }
                }else{
                    System.out.println("对不起，你输入的信息有误！");
                }
            }
        }
    }

    /**
     * 取款功能
     *
     * @param acc
     * @param sc
     */
    private static void drawMoney(Account acc, Scanner sc) {
        System.out.println("=================用户取钱功能=================");
        //1.判断账户是否足够100元
        if (acc.getMoney() < 100) {
            System.out.println("当前账户余额不足100元，不能取钱。");
            return;
        }
        while (true) {
            //2.提示用户输入取款金额
            System.out.println("请你输入取款额度");
            double money = sc.nextDouble();

            //3.判断金额是否超出存款
            if (acc.getMoney() >= money) {
                if (acc.getQuotaMoney() < money) {
                    System.out.println("你取款的额度大于当次限额" + acc.getQuotaMoney());
                } else {
                    acc.setMoney(acc.getMoney() - money);
                    System.out.println("你取款" + money + "成功！");
                    showAccount(acc);
                    return;
                }
            } else {
                System.out.println("你的额度不足，当前额度：" + acc.getMoney());
            }
        }
    }

    /**
     * 存钱
     *
     * @param acc

    private static void depositMoney(Account acc, Scanner sc) {
        System.out.println("===============用户存钱操作===============");

        //更新账户余额原来的钱加上存入的钱

    }

    /**
     * 方法用户开户功能的实现
     *
     * @param accounts 接收的账户集合
     */
    private static void register(ArrayList<Account> accounts, Scanner sc) {
        System.out.println("===============系统开户操作===============");
        //1.创建一个账户对象，用于后期封装账户信息。
        Account account = new Account();

        //2.录入当前账户信息
        System.out.println("请输入账户用户名");
        String userName = sc.next();
        account.setUserName(userName);

        while (true) {
            System.out.println("请输入账户密码");
            String passWord = sc.next();
            account.setUserName(userName);
            System.out.println("请确认账户密码");
            String okpassWord = sc.next();
            account.setUserName(userName);
            if (passWord.equals(okpassWord)) {
                //密码认证通过
                account.setPassWord(passWord);
                break;
            } else {
                System.out.println("你输入的两次密码不同，请重新确认！");
            }
        }
        System.out.println("请输入账户当次限额：");
        double qutaMoney = sc.nextDouble();
        account.setQuotaMoney(qutaMoney);
        //为账户随机8位账户卡号(独立功能独立成方法)
        String cardId = getRandCardId(accounts);
        account.setCarId(cardId);

        //3.把账户对象添加到账户集合中去
        accounts.add(account);
        System.out.println("恭喜" + userName + "开户完成" + "你的卡号是：" + cardId + "，请妥善保管");
    }

    /**
     * 生成8位随机卡号
     *
     * @return
     */
    private static String getRandCardId(ArrayList<Account> accounts) {
        //1.生成8位数字
        Random r = new Random();
        while (true) {
            String carId = "";
            for (int i = 0; i < 8; i++) {
                carId += r.nextInt(10);
            }
            //2.判断这个8位的卡号是否和其他账户相同
            Account acc = getAccountByCarId(carId, accounts);
            if (acc == null) {
                return carId;
            }
        }
    }

    /**
     * 验证呢卡号是否重复
     *
     * @param carId
     * @param accounts
     * @return
     */
    private static Account getAccountByCarId(String carId, ArrayList<Account> accounts) {
        for (int i = 0; i < accounts.size(); i++) {
            Account acc = accounts.get(i);
            if (carId.equals(acc.getCarId())) {
                return acc;
            }
        }
        return null;//查无此用户
    }

案例：打印整型数组内容

  1. 需求：设计一个方法用于输出任意整型数组的内容，要求输出成如下格式：
    该数组内容为：[11,22,33,44,55]
  2. 分析：
    • 定义一个方法，要求该方法能够接收数组，并输出内容。 —> 需要参数吗？返回值需要声明吗?
    • 定义一个静态化初始数组，调用该方法，并传入该数组。
  3. 代码实现：
    package param;

public class Text3 {
    public static void main(String[] args) {
        // 需求：设计一个方法用于输出任意整型数组的内容
        int[] arr1 = {11,22,33,44};
        getArray(arr1);
        System.out.println("------------------");
        int[] arr2 = {10,20,30,40};
        getArray(arr2);
        System.out.println("---------------");
        //特殊时候，传递空值
        int[] arr3 = null;
        getArray(arr3);

    }
    public static void getArray(int[] arr){
        System.out.print("[");
       if (arr != null && arr.length > 0){
           for (int i = 0; i < arr.length; i++) {
               //1.if写法
               if (i == arr.length -1) {
                   System.out.print(arr[i]);
               }else {
                   System.out.print(arr[i] + ",");
               }
               //
               //2.三元运算符写法
               System.out.print(i == arr.length - 1 ? arr[i]:arr[i] + ",");
               //
           }
       }
        System.out.println("]");
    }
}

案例：从数组中查询指定元素的索引

  1. 需求：设计一个方法可以接收整型数组，和要查询的元素值；最终要返回元素在该数组中的索引，如果数组中不存在该元素则返回。
    示例： 
    例如：[11,22,33,44,55]
输入元素：44，返回索引3
输入元素：88，返回-1
  2. 分析：
    • 定义方法，接收整型数组，查询的元素值，在方法体中完成元素查询的功能。 —–> 是否需要参数、返回值类型？
    • 定义数组，调用该方法，并指定要搜索的元素值，得到返回的结果输出。
  3. 代码实现：
    package param;

import java.util.Scanner;

public class Text4 {
    public static void main(String[] args) {
        //需求：设计一个方法可以接收整型数组，和要查询的元素值；最终要返回元素在该数组中的索引

        //3.定义一个数组
        int[] arr1 = {11,22,33,44};
        int index = getArray(arr1,44);
        System.out.println("您查询的元素的索引是：" + index);
    }
    //1.定义一个方法
    public static int getArray(int[] arr,int date){

        //2.开始找出这个数据的索引
        for (int i = 0; i < arr.length; i++) {
            if(arr[i] == date){
                return i;
            }
        }
        return -1;//查无此元素
    }
}

案例：比较两个数组的内容是否一样

  1. 需求：如果两个数组的类型，元素个数，元素顺序和内容是一样的我们就认为这两个数组是一摸一样的。 
    例如：如下两个数组是一样的
int[] arrs1 = {10,20,30};
int[] arrs2 = {10,20,30};
    请用方法完成：能够判断任意两个整型数组是否一样，并返回true或者false。
  2. 分析：
    • 定义方法，接收2个整型数组 —> 是否需要参数，返回值类型？
    • 在方法内部完成判断的逻辑，并返回布尔结果。
  3. 代码实现：
    package param;

public class Text5 {
    public static void main(String[] args) {
        //需求：比较两个数组的内容是否一样
        int[] arr1 = {11,122,33};
        int[] arr2 = {11,22,33};
        System.out.println(getArray(arr1,arr2));
    }
    public static boolean getArray(int[] arr1,int[] arr2){
        //判断两端数组长度是否相等
        if (arr1.length == arr2.length) {
            for (int i = 0; i < arr1.length; i++) {
                //判断数组元素是否相同
                if(arr1[i] != arr2[i]){
                    return false;
                }
            }
            //这个返回值要在循环完之后返回
            return true;
        }
        return false;
    }
}

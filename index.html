    （1）面向对象编程

/**
 * 生成8位随机卡号
 *
 * @return
 */
private static String getRandCardId(ArrayList<Account> accounts) {
    //1.生成8位数字
    Random r = new Random();
    while (true) {
        String carId = "";
        for (int i = 0; i < 8; i++) {
            carId += r.nextInt(10);
        }

        //2.判断这个8位的卡号是否和其他账户相同
        Account acc = getAccountByCarId(carId,accounts);
        if (acc == null){
            return carId;
        }
    }
}

/**
 * 验证呢卡号是否重复
 * @param carId
 * @param accounts
 * @return
 */
private static Account getAccountByCarId(String carId, ArrayList<Account> accounts) {
    for (int i = 0; i < accounts.size(); i++) {
        Account acc = accounts.get(i);
        if (carId.equals(acc.getCarId())) {
            return acc;
        }
    }
    return null;//查无此用户
}

集合概述、ArrayList的使用

集合概述

  1. 集合与数组类似，也是一种容器，用于装数据的。
  2. 数组的特点
    • 数组定义完成启动之后，类型确定，长度固定。
    • 问题：在个数不能确定，且要进行增删数据操作的时候，数组是不大合适的。
  3. 集合的特点
    • 集合的大小不固定，启动之后可以动态变化，类型也可以选择不固定。
    • 集合非常适合做元素个数不确定，且要进行增删操作的业务场景。
    • 集合的提供了很多功能，而数组功能单一。
  4. 总结
    （1）数组和集合的元素储存的个数问题？
    • 数组定义后类型确定，长度固定
    • 集合类型可以不固定，大小是可变的。

String概述

  • String其实常被称为不可变字符串类型，它的对象在创建后不能被更改。
  • 示例

  1. 字符串对象存在哪里？
  • 以””方式给出的字符串对象，在字符串常量池中储存。

    （1）String是什么，可以做什么？
    • 字符串类型，可以定义字符串类型变量指向字符串对象。
      （2）String是不可变字符串的原因？
    • String变量每次的修改都是产生并指向新的字符串对象。
    • 原来的字符串对象都是没有改变的，所以称不可变字符串。

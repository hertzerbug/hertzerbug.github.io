表达式变量类型

Posted on 2022-07-13 Edited on 2022-07-14
  1. 小范围的类型会自动转换成大范围的类型运算。
  2. 最终类型由表达式中最高类型决定。
  3. byte short char 是直接转换成int类型参与运算的。
    package com.itheima.type;

public class TypeDome {
    public static void main(String[] args) {
        //表达式的自动类型转换
        byte a = 10;
        int b = 20;
        double c = 1.0;
        double rs = a+b+c;//转成double类型
        System.out.println(rs);

        double rs2 = a+b -2.3;//转成double类型
        System.out.println(rs2);

        //面试题
        byte i = 10;
        byte j = 20;
        int k = i+j;//转成int类型
        System.out.println(k);

    }
}

自动类型转换

Posted on 2022-07-13 Edited on 2022-07-14

自动类型转换

byte–short–int–long–float–double

package com.itheima.type;

public class TypeDemo1 {
    public static void main(String[] args) {
        //自动类型转换
        //类型范围小的变量可以直接赋值给类型范围大的变量
        byte a = 20;
        int b = a ;
        System.out.println(a);
        System.out.println(b);


        int age = 23;
        double db = age ;
        System.out.println(db);//结果为23.0
        
    }
}
